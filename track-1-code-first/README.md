# Track 1: Code-First Design System

En este proyecto exploramos la creación de un Design System puramente en código, enfocado en prototipar rápidamente sin depender de diseños previos en Figma.

## Objetivo
- Desarrollar componentes agnósticos o en el framework seleccionado (actualmente HTML/CSS puro).
- Prototipar vistas funcionales consumiendo estos componentes que existen únicamente aquí.

## Siguientes Pasos (Sugeridos)
- [ ] Definir variables base (colores, tipografía y espaciados básicos en CSS).
- [ ] Construir los primeros componentes atómicos (botones, tarjetas, inputs).
- [ ] Ensamblar una vista de prueba o prototipo inicial.
